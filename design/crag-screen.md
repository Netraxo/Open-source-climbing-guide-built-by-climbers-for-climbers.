# Crag Screen

## Purpose

The Crag Screen provides all essential information about a climbing crag.

It should help climbers quickly decide whether the crag matches their plans for the day.

---

## User Goal

Learn about the crag and choose the best sector.

---

## Success

The user selects a sector within seconds.

---

# Layout

Header

- Crag name
- Rock type

Hero Image

Main photo of the crag.

Information

- Sport routes
- Trad routes
- Number of sectors
- Maximum height
- Difficulty range
- Sun exposure
- Approach time
- Parking
- Child friendly

Description

Short description of the crag.

Sector List

Every sector displays:

- Sector name
- Number of routes
- Difficulty range
- Maximum height

Map Button

Open the sector directly on the map.

# Design Principles

- Show only essential information.
- Keep the page clean.
- Minimize scrolling.
- Allow quick access to sectors.
